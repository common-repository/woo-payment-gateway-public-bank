== Description ==

Allow woocommerce pay with credit card of Public bank using CyberSource method.
Support Installment method.


== Installation ==

= Installing The Payment Gateway Plugin =
* Download the plugin zip file.
* Login to your WordPress Admin. Click on Plugins | Add New from the left hand menu.
* Click on the “Upload” option, then click “Choose File” to select the zip file from your computer. Once selected, press “OK” and press the "Install Now" button.
* Activate the plugin.
* Open the settings page for WooCommerce and click the "Payment Gateways," tab.
* Click on the sub tab for "Public Bank."
* Configure your settings.

__ Notice that some advance options:__
__test mode: when it is enable user can use test credit card to pay reservation. It only for use in testing.
__debug: Allow the plugin write log for each transaction.(Recommend turn ON)

== Frequently Asked Questions ==

= Do I need a merchant account before I can use the plugin? =
Yes.  In order to use this plugin you will need a merchant services account. Checkout https://www.pbebank.com/ and contact sale or support for more information.
